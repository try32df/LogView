# LogView

**LogView** — графическая Linux-утилита для просмотра и фильтрации логов.

Программа позволяет:
- просматривать стандартные Linux-логи;
- открывать собственные `.log` и `.txt` файлы;
- выводить последние строки логов;
- фильтровать строки по тексту;
- сохранять результат в `.txt`;

---

# Установка зависимостей

Войти в root:

```bash
su
```

Установить необходимые пакеты:

```bash
apt update
apt install python3 python3-tk git -y
```

Выйти из root:

```bash
exit
```

# Скачивание проекта

Скачать проект:

```bash
git clone https://github.com/USERNAME/LogView.git
```

Перейти в папку проекта:

```bash
cd LogView
```

# Выдача права на запуск

После загрузки через сайт GitHub Linux может убрать право на запуск файла.

Поэтому обязательно выполнить:

```bash
chmod +x LogView
```

# Возможности программы

## Просмотр Linux логов
Можно быстро открыть системные Linux-логи.

## Открытие своего файла
Можно выбрать любой `.txt` или `.log` файл.

## Фильтрация текста
Можно искать ошибки, слова или нужные строки.

## Сохранение результата
Вывод можно сохранить в отдельный `.txt` файл.

## GUI интерфейс
Программа работает через графический интерфейс.
