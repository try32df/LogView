# LogView

**LogView** — графическая Linux-утилита для просмотра и фильтрации логов.

Программа позволяет:
- просматривать системные Linux-логи;
- открывать собственные `.txt` и `.log` файлы;
- выводить последние строки файлов;
- искать нужный текст через фильтр;
- сохранять результат;
- работать через красивый GUI интерфейс;
- закрывать программу через кнопку **Выход** или кнопку **✕**.

---

# Установка зависимостей

Войти в root:

```bash
su
```

Установить необходимые пакеты:

```bash
apt update
apt install python3 python3-tk git -y
```

Выйти из root:

```bash
exit
```

# Скачивание проекта

Скачать проект:

```bash
git clone https://github.com/USERNAME/LogView.git
```

Перейти в папку проекта:

```bash
cd LogView
```

---

# ВАЖНО

Если проект был загружен на GitHub через сайт, Linux может убрать право на запуск файла `LogView`.

Поэтому ОБЯЗАТЕЛЬНО вручную написать команду:

```bash
chmod +x LogView
```

⚠️ Команду нужно ПЕЧАТАТЬ вручную.  
Не копировать старые строки из терминала.

---

# Запуск программы

После этого программа запускается так:

```bash
./LogView
```

---

# Если появляется ошибка "команда не найдена"

Значит команда была введена с лишними символами.

Нужно вручную написать:

```bash
cd ~/LogView
chmod +x LogView
./LogView
```

# Возможности программы

## Просмотр Linux логов
Можно быстро открыть системные Linux-логи.

## Открытие собственного файла
Можно выбрать любой `.txt` или `.log` файл.

## Фильтрация текста
Можно искать ошибки, предупреждения или нужные строки.

## Сохранение результата
Результат можно сохранить в отдельный `.txt` файл.

## GUI интерфейс
Программа работает через графический интерфейс.
